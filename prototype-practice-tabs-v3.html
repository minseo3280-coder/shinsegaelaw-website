G-1 — 1단: 골드 라인 / 2단: 버건디 라인

1단 하단 라인이 골드. 2단은 기존 버건디. 두 단의 역할이 색으로 명확히 구분됨.

BANNER

G-2 — 1단: 네이비 라인 / 2단: 버건디 라인

1단 하단 라인이 네이비(#1A1A2E). 무게감 있고 차분. 영문은 골드 유지.

BANNER

G-3 — 1단: 버건디→골드 그라데이션 라인 / 2단: 버건디

1단 하단 라인이 버건디에서 골드로 전환. 사이트 곳곳에서 쓰이는 시그니처 그라데이션.

BANNER

G-4 — 1단: 골드 라인 + 골드 도트 / 2단: 버건디

골드 라인에 추가로 활성 탭 앞에 작은 골드 도트. 한 눈에 어디인지 포인트.

BANNER

G-5 — 1단: 라인 없이 배경 틴트 / 2단: 버건디

1단 활성 탭에 버건디 틴트 배경(라운드). 라인 대신 면적으로 구분. 부드러운 느낌.
